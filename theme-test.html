Portfolio Redesign

Personal Project

Objective

In April of 2018, I designed and coded a website for the first time. Determined to learn more about HTML and CSS, I set out to create the whole thing from scratch, and quite honestly I fell in love the whole process of designing for the web. Even once I finished creating my portfolio website, I wanted to learn more, and chased every opportunity to keep practicing and experimenting.

Of course, six months of improving my web skills left me feeling that my initial website design needed some tuning up, so I set a few goals for the redesign:

1. Improve accessibility and general organization.

2. Alter the color scheme to something more inviting.

3. Reorganize and improve imagery, where needed.

4. Add subtle interactivity on supporting pages.

5. General updates of content, where needed.

For the previous project in this class, I had worked with LINK: Mrs. Eaves, so I aimed to make this project about something a little less traditional: Futura.

Experimentation

Especially in my student works, I try to make a point of pushing outside of what I'm comfortable with in design. I don't really believe you can improve without pushing past what you're used to, and in my own work I have always seen marked improvement after an unusual or ambitious project. This zine was certainly no exception to my trend of stepping outside my own lines, including extremely new to me techniques like costume-making and long-exposure photography.

However, this push into the unknown was only fitting for a typeface meant to embody futurism so strongly, and that futuristic theme resonated throughout the piece, from light-writing to an astronaut suit.
