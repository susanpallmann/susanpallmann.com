Project Page Generator

Not Quite a CMS

This entire website is coded from scratch - HTML, CSS, and JavaScript. Although it might be simpler and faster to use a website builder or Content Management System (CMS), a from-scratch approach provides virtually unlimited customization, and a fantastic opportunity to deepen one's knowledge of web design and development.

That being said, a programmer able to code an entire website from scratch should probably also be able to code a tool to make creating future pages a little less time-intensive. This page generator is for adding new project (portfolio pieces) pages.
